# [SomeBeerApp](https://my420.github.io/SomeBeerApp/)

##### SPA, [React](https://reactjs.org/), [Redux](https://redux.js.org/), [Immutable](https://immutable-js.github.io/immutable-js/), [React Router](https://github.com/ReactTraining/react-router), [Create React App](https://facebook.github.io/create-react-app/), [Jest](https://jestjs.io/), [Enzyme](https://airbnb.io/enzyme/), [PUNK API](https://punkapi.com/).
SomeBeerApp – это импровизированный онлайн магазин крафтового пива. В качестве бэкенда использован [Punk Api](https://punkapi.com/).

* HTML5, CSS3.
* Используемый препроцессор: Sass.
* Адаптивная, семантическая, валидная вёрстка. 
* Используемый инструмент автоматизации: Create React App.
* Основные библиотеки: React, Redux, React Router, **Immutable**.
* Дополнительные библиотеки: **CSS Modules**, **ReactTransitionGroup**, **Redux-Thunk**, querystringify.
* Библиотеки для тестирования: **Jest**, **Enzyme**.
* **Airbnb JavaScript Style Guide**.

#### Особенности:
Реализованы стандартные возможности интернет магазинов:
1. Добавление, удаление товаров из корзины.
2. Изменение количества товаров в корзине.
3. Отображение количества товара, добавленного в корзину, рядом с корзиной.
4. Добавление, удаление товаров в "избранные". Избранные товары хранятся в **localStorage**.
5. На главной странице реализован **слайдер**.
6. **Поиск** по каталогу товаров с дополнительными параметрами.
7. Отображение **популярных товаров**. Список меняется каждый день.
8. **Случайный** выбор товара, реализованный в виде **рулетки**.
